# 研究與專題經驗

## AIoT 即時遠端設備操作系統
**研究領域**：AIoT / 即時通訊 / 遠端控制系統  

本研究設計一套結合人工智慧與物聯網的遠端操作平台，透過即時影像分析與設備控制模組，使使用者能夠在遠端環境中即時操作設備。

### 技術重點
- YOLO 物件偵測模型  
- 即時影像串流  
- 遠端設備控制系統  
- IoT架構設計  

### 研究目標
- 降低遠端操作延遲  
- 提升系統即時回饋能力  
- 建立可擴展的AIoT應用架構
